# Leo VI's Mass Vassal Directives. Exempt marker, by-name override.
#
# Redefines two of vanilla's vassal-directive customizable-localization
# functions: vassal_directive_icon and vassal_directive_text. Each vanilla text
# entry gets a preceding twin with the same trigger plus
# has_variable = leo_mvd_exempt, pointing at a leo_mvd_x_* key that renders the
# exempt version (a gray icon, or the text plus an exempt note).
#
# Customizable-localization functions override by name, last loaded wins, so
# this file does not replace vanilla's 00_vassal_custom_loc.txt. It redefines
# only these two functions and leaves the rest of that file, and any future
# vanilla additions to it, in place. The zz_ prefix loads it last.
#
# Re-diff these two functions against vanilla's 00_vassal_custom_loc.txt after
# every game patch. A new directive means new twin entries here.
